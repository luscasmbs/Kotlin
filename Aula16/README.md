# Dia 16 - Casos Extremos em Busca e Integração com Estruturas de Dados

## 📚 Conteúdo da Aula
Neste dia estudamos casos extremos em algoritmos de busca e integração com estruturas de dados, principalmente utilizando listas e Map em Kotlin.

### Assuntos abordados
- Busca linear
- Busca binária
- Tratamento de casos extremos
- Listas vazias
- Valores inexistentes
- Valores repetidos
- Integração com `Map`
- Complexidade de algoritmos:
  - O(n)
  - O(log n)
  - O(1)

---

## ⚠️ Casos extremos estudados
- Lista vazia
- Lista com apenas 1 elemento
- Elemento inexistente
- Índices inválidos
- Valores repetidos
